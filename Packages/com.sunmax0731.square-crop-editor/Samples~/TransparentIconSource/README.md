# Transparent Icon Source Sample

This sample contains a transparent PNG source image for manual smoke testing.

## Test Steps

1. Import this sample from Unity Package Manager.
2. Open `Tools > Square Crop Editor > 繝｡繧､繝ｳ逕ｻ髱｢`.
3. Assign `square-crop-sample.png` to `Source Image`.
4. Drag a crop region around one of the colored shapes.
5. Change `Output Ratio` and `Mapping`.
6. Export to `Assets/Generated/SquareCrop`.
7. Confirm the exported PNG keeps transparent background pixels.
